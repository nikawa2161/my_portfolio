# サイト概要
「転職をもっと気軽に」
企業側と求職者のマッチングを助けるアプリです。

# URL
http://re-life.work/
「閲覧用」から企業側・ユーザーの簡単ログインが可能です。

# サイトテーマ
リライフ（relife）の意味は「再び」　「rehabilitation　life（リハビリライフ）」を掛け合わせました。
「リハビリ職の人生をよりよくする」がテーマです。

# テーマを選んだ理由
現職の理学療法士は、２０３０年以降から供過多になると言われています。
